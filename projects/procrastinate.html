Overview

Built at TAMUhack, where it won Best Startup. ProcrastiNATE is a Chrome extension that integrates with Canvas to gamify assignment tracking, addressing the common problem of deadlines getting lost inside the platform.

It uses AWS Comprehend to extract key phrases and deadlines directly from course pages, converting them into reminders and lightweight game mechanics.
